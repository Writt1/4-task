# Парсер

- **IB_ADD_ADS_20251011153000.php**  
  Миграция (модуль sprint.migration), которая создаёт(если еще не существуют):
  - Тип инфоблока «Объявления»
  - Инфоблок «Недвижимость»  

- **realestate.csv**  
  Файл с данными для импорта.

- **realestate.php**  
  Скрипт, который импортирует данные из realestate.csv в инфоблок (перед запуском нужно указать актуальный ID инфоблока в переменной `$IBLOCK_ID`).
